Wisconsin State Cartographer's Office Control Finder Application

Another responsibility while working for the Wisconsin State Cartographer's Office, was helping to maintaining the Control Finder database that updates all of Wisconsin's National Geodic Survey Data. The application (not my design), has also since been redesigned and can be viewed by clicking on the image above. The workflow for this was slightly more intensive, specifically in using python to download and parse data into their internal structure for the application.
