Wisconsin Hockey Rink and WAHA District Webmap

Current work in progress for the Wisconsin Amateur Hockey Association

About

This interactive map details all rinks and associations listed on the WAHA website. Currently there is not a published map of WAHA regions and districts, but instead a collection of PDFs detailing districts, counties, and cities/townships that make up each region that WAHA oversees. This is NOT to be used as official WAHA communications or rules. Instead it is an open attempt at working with WAHA to create a visual representation of youth hockey in the state of Wisconsin. Please see map description for further notes and resources.
