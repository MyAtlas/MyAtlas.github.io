Wisconsin State Cartographer's Office Coastal Bordner Survey Explorer Application

While working for the Wisconsin State Cartographer's Office, I had the opportunity of helping prepare 1930's aerial photographs for an interactive application highlighting Bordner Survey Map features. The workflow included pre-processing the raw images into folders sorted by township using a python script. From there we used a photoshop script to stitch those images together. Next we manually georeferenced the townships before finally mosaicking them and clipping them by county to be added to the final map tiles. To view the application, click the image above.
